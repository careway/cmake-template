# CMake C++ Template 

Cmake c++ template project, that brings cpack and automatic versioning based on git.

## Features

- [x] Cpack
- [x] Versioning through git, aplied to doxygen and cpack
- [x] Generates version.h 

## Replace Variables

To customize the project replace the following words with the desired final name

- RPPROJECTNAME
- RPALIAS
- RPLICENSERPM
- RPCPACKDESCRIPTION
- RPCPACKVENDOR
- RPCPACKCONTACT
- RPLOGOPATH

Remember please to rename the `cmake/rpprojectnameConfig.cmake`.
